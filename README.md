# Poopa Scoopa

**Taking the dirty work out of pet ownership**

Smart tech for crappy situations.

## About

Poopa Scoopa is a modern web application built with Vite and React. Where cutting-edge robotics meet pet care - a robot that handles the messy stuff, so you can enjoy the furry cuddles.

## Tech Stack

- **Vite** - Fast build tool and dev server
- **React** - UI framework
- **Modern CSS** - Clean, responsive design

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (typically http://localhost:5173)

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Design Philosophy

- **Visual Style**: Clean, modern design with rounded corners and friendly shapes
- **Color Palette**: Earth tones (browns, greens) with pops of bright, cheerful colors like yellow or orange
- **Tone**: Witty but straightforward - acknowledge the silly name while emphasizing the practical solution
- **Animations**: Smooth animations showing the robot in action
- **White Space**: Plenty of white space to feel fresh and sanitary
